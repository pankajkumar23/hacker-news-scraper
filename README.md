# HackerNews Scraper 🔥

Scrapes top news from HackerNews with title, link and score.

## What it does
- Scrapes top 10 news from news.ycombinator.com
- Saves Title, Link and Score in CSV file

## Tech Used
- Python
- BeautifulSoup
- Requests
